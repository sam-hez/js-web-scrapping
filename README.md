## What it does

This project shows how to use JavaScript to get information from web pages.
It now includes updated scraping logic and clearer output handling.

## How to use

1. Open the project folder.
2. Install dependencies if needed: `npm install`.
3. Run the scraper file with Node.js: `node scraper.js`.
4. Check the console output for the scraped results.

## How to test

1. Make sure Node.js is installed.
2. Run the scraper in the terminal: `node scraper.js`.
3. Verify the output matches the expected page data.
4. If the project writes results to a file, open the generated output file and confirm the values.
